# mastodon-client

Mastodon client project.

## Cursor Cloud Agents

- Dashboard: [cursor.com/agents](https://cursor.com/agents)
- API key: [cursor.com/settings](https://cursor.com/settings) → set `CURSOR_API_KEY`, then run `./scripts/create-cloud-agent.sh`
- In-editor MCP: `.cursor/mcp.json` (uses `cursor-cloud-agent-mcp` and the same env var)
